# Seneca Weekly Walkthroughs

Dynamic web slide decks for Seneca courses, a companion to Blackboard. These
replace the traditional PowerPoint walkthrough with an interactive, brand-skinned
deck built on Reveal.js.

Open `index.html` for the gallery. Each deck: arrow keys or swipe to move, click
to reveal, click a key-term card to flip, press `S` for the narration view, and
append `?print-pdf` to export a PDF.

This is the public, copyright-clean build: slides are text-first, with no hosted
third-party images. Source decks are generated from course YAML by
`web_builder.py` in the Seneca course system. Decks are set to `noindex`.
